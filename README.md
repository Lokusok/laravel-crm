## CRM-система на Laravel

- Laravel
- Minio S3
- Redis
- Mailpit
- MySQL
- Docker

---

## Запуск проекта

**Предварительно, установите Composer: https://getcomposer.org/download/**

1. `make install`
2. `make start`
3. `make setup`
